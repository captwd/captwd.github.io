给计算机新生的建议

发表于2026-07-08|更新于2026-07-09
|浏览量:

呃，我简单讲一些，给一些方向吧。

基础技能

  • 简单PPT制作
  • 简单Word用法 — 小组作业靠你了
  • 简单Markdown编写 — 有官网可以看看
  • 一些文件类别相互转换的方法
  • 一个写简历的网站

编程工具

  • 一个好的编程IDE：VSCode 配置操作比较麻烦，跟着 AI 走总能配好，最后我好像是直接改了那个配置文件弄好的
  • 了解智能体，学会用 AI 编程工具：Trae、Cursor 这类 AI 编程 IDE，或者在 VS Code 里配置一下 Claude Code。具体方法 B 站上都有很多，我比较喜欢 Skill，感觉能做很多事

开发工具

  • 一些简单的 Git 使用
  • 一些简单的 Linux 命令

其他

  • 谷歌邮箱很方便，能做很多事
  • 学会逛逛 GitHub、CSDN（虽然差评很多）等
